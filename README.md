# 🧠 Visual Memory Test

เกมทดสอบความจำภาพ ผู้เล่นต้องจดจำตำแหน่งของช่องสีน้ำเงินที่ปรากฏบน Grid แล้วกดเลือกให้ถูกต้องก่อนที่ Pattern จะหายไป แต่ละ Level จะเพิ่มจำนวนช่องและขนาด Grid ให้ยากขึ้นเรื่อยๆ จนถึง Level 10

**กลุ่มเป้าหมาย:** เด็กอายุ 8–12 ปี

---

## วิธีเล่น

1. กด **Start Test** เพื่อเริ่มเกม
2. จดจำ **ตำแหน่งช่องสีน้ำเงิน** บน Grid ให้ได้ก่อนที่มันจะหายไป
3.  เมื่อช่องสีน้ำเงินหายไป ให้กดเลือก **ตำแหน่งที่เคยเห็น** ให้ครบถ้วน
4. ถ้าเลือกถูกต้องทั้งหมด → **ผ่านไปยัง Level ถัดไป**
5. ถ้าเลือกผิด → **เสีย 1 Lives** หากหมด Lives เกมจบ
